ทำความเข้าใจ AI Agent, 4 Components และ Model Context Protocol ก่อนลงมือปฏิบัติ

AI ธรรมดา vs AI Agent

🤖 AI ธรรมดา

ถาม → ตอบ → จบ
ตอบจากความรู้เดิม
ทำได้แค่ครั้งเดียว
ไม่รู้ข้อมูลปัจจุบัน

⚡ AI Agent

รับเป้าหมาย → วางแผน → ทำงาน → วนซ้ำ
ใช้ tools ดึงข้อมูลจริง
ทำงานหลายขั้นตอน
แจ้งเตือน ส่ง email ได้เอง

4 ส่วนประกอบของ AI Agent

🧠
LLM — สมอง
ตัดสินใจและวางแผน (Claude, GPT-4, Gemini)

LLM (Large Language Model) คือหัวใจของ Agent ทำหน้าที่คิดวิเคราะห์ เลือกว่าจะใช้ Tool ไหน และตอบคำถาม

Claude 3.5 Sonnet เหมาะที่สุดสำหรับงาน CPF เพราะอ่านภาษาไทยได้ดีมาก
🔧
Tools — มือ
เครื่องมือที่ Agent ใช้ทำงานจริง
  • 1เว็บ/API — ดึงราคาหุ้น, อัตราแลกเปลี่ยน, ข่าวสาร
  • 2ไฟล์ — อ่าน Excel, PDF, Word ได้
  • 3Email/LINE — ส่งแจ้งเตือนอัตโนมัติ
  • 4Database — ค้นหาและบันทึกข้อมูล
Tools ทำงานแทนเรา — Agent เป็นคนสั่ง Tools เป็นคนทำ
💾
Memory — ความจำ
จดจำบริบทและประวัติการสนทนา

Agent จำเนื้อหาที่คุยกันก่อนหน้า ทำให้ไม่ต้องพิมพ์ context ซ้ำทุกครั้ง

Claude Projects ใน claude.ai คือ Memory ที่ง่ายที่สุด — อัปโหลดเอกสารครั้งเดียว ใช้ได้ตลอด Project
🔁
Planning Loop — วงจรการทำงาน
คิด → ทำ → ดูผล → ทำต่อ จนสำเร็จ
AI Agent Loop
คิด: ต้องดูข้อมูล QC วันนี้ก่อน ↓ ทำ: [เรียก Tool: อ่าน QC_Report.xlsx] ↓ ดูผล: พบ 2 LOT ไม่ผ่าน (Line 2) ↓ คิด: ต้องแจ้งเตือน QC Manager ↓ ทำ: [เรียก Tool: ส่ง LINE Notify] ↓ คิด: เสร็จแล้ว → จบการทำงาน

MCP — Model Context Protocol

🔌 MCP คืออะไร? (อธิบายแบบง่าย)

MCP คือ มาตรฐานกลาง สำหรับให้ AI เชื่อมต่อกับเครื่องมือต่างๆ เปรียบเหมือน USB-C ของ AI — เสียบได้ทุกอุปกรณ์

❌ ก่อนมี MCP
ต้องเขียน code เชื่อมแต่ละ tool แยกกัน
เปลี่ยน AI ต้องเขียนใหม่ทั้งหมด
✅ หลังมี MCP
เชื่อมครั้งเดียว AI ทุกตัวใช้ได้
500+ MCP Servers พร้อมใช้ฟรี
🔌
MCP ที่ใช้ใน Workshop นี้
ตัวอย่าง MCP Servers ที่ CPF ใช้งานได้เลย
  • 1Gmail MCP — AI อ่านและส่งอีเมลได้
  • 2Google Sheets MCP — AI ดึงข้อมูล QC จาก Sheets
  • 3Google Drive MCP — AI เข้าถึงเอกสาร
  • 4Filesystem MCP — AI อ่านไฟล์บนเครื่อง
  • 5n8n MCP — Workflow ของ n8n กลายเป็น Tools ให้ AI ใช้
n8n มี MCP Node ในตัว — Workflow ที่สร้างใน n8n สามารถเป็น Tool ให้ Claude ใช้ได้ทันที
