ภาพรวม Dify

✨ Dify คืออะไร?

Dify เป็น platform open-source สำหรับสร้าง AI Chatbot จากเอกสารของเราเอง (RAG - Retrieval Augmented Generation) อัปโหลดเอกสาร HR, คู่มือผลิตภัณฑ์ หรือ SOP แล้ว Chatbot จะตอบจากเอกสารนั้นๆ โดยตรง

  • 1กด Knowledge ในเมนูซ้าย
  • 2กด + Create Knowledge
  • 3ตั้งชื่อ: CPF HR Policy 2569

  • 5Chunking: เลือก Automatic

4
ใส่ System Prompt
บอก AI ว่าตัวเองคือใครและทำหน้าที่อะไร
📋 System Prompt — CPF HR Assistant 
คุณเป็น HR Assistant ของ CPF ชื่อ "น้องซีพี"
ช่วยเหลือพนักงานโรงงานด้านสวัสดิการและระเบียบบริษัท

หน้าที่:
✅ ตอบคำถามสวัสดิการ วันลา OT เงินเดือน
✅ อธิบายระเบียบบริษัทให้เข้าใจง่าย
✅ แนะนำขั้นตอนการยื่นเรื่องต่างๆ

กฎสำคัญ:
📌 ตอบจากเอกสาร HR Policy เท่านั้น
📌 ถ้าไม่มีข้อมูล → แนะนำติดต่อ HR: 02-XXX-XXXX
📌 ใช้ภาษาไทยเสมอ พูดง่ายๆ เหมือนเพื่อนร่วมงาน
📌 ห้ามเดาหรือตอบข้อมูลที่ไม่แน่ใจ

เปิดด้วย: "สวัสดีครับ ผมน้องซีพี ช่วยตอบคำถาม HR ได้เลย 😊"
5
เชื่อม Knowledge Base + ตั้งค่า
เชื่อมเอกสาร HR และปรับ AI settings
  • 1ส่วน Context กด + Add
  • 2เลือก CPF HR Policy 2569
  • 3Top K: 5 (ดึงข้อมูลที่เกี่ยวข้อง 5 ส่วน)
  • 4Score Threshold: 0.5
  • 5เปิด Citation ✅ (แสดงแหล่งอ้างอิง)
  • 6Temperature: 0.3 (ตอบสม่ำเสมอ)
เปิด Citation เสมอ — พนักงานจะเห็นว่าตอบจากส่วนไหนของ HR Policy
6
ทดสอบ Chatbot
ลองถามคำถามที่พนักงานถามบ่อย
👤 พนักงาน
OT วันอาทิตย์ได้เงินกี่เท่า?
🤖 AI Assistant
วันหยุดประจำสัปดาห์ได้ 3 เท่าของค่าจ้างรายชั่วโมงครับ 💰

ตัวอย่าง: เงินเดือน 15,000 บาท → ชั่วโมงละ ~71 บาท → OT = 213 บาท/ชั่วโมง

📋 อ้างอิง: นโยบาย OT หน้า 3
👤 พนักงาน
ลาป่วย 2 วันต้องมีใบแพทย์ไหม?
🤖 AI Assistant
ลาป่วย 1-2 วัน ไม่ต้องมีใบแพทย์ครับ ✅
แต่ต้องแจ้งหัวหน้างานก่อน 08:00 น. ของวันนั้น

ถ้าลา 3 วันขึ้นไป ต้องมีใบรับรองแพทย์ครับ 😊

Publish และแชร์
3 วิธีให้พนักงานเข้าใช้งาน

  • 2เลือก Web App → ได้ Link
  • 3สร้าง QR Code จาก Link → พิมพ์ติดบอร์ดประกาศ
  • 4หรือแชร์ Link ทาง LINE กลุ่มพนักงาน
  • 5ทดสอบ Link ในโหมด Incognito ก่อนแชร์
