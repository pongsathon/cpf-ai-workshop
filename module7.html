Capstone Project — สร้าง AI Agent ประจำแผนก

📋 ขั้นตอนการทำ Capstone

⏰ เวลา: 60 นาที (ทำงานเป็นกลุ่ม 3-4 คน)
🎯 เป้าหมาย: สร้าง AI Agent จริงที่ใช้งานได้หลัง Workshop
🏆 Showcase: นำเสนอ 5 นาทีต่อกลุ่ม
🏭
กลุ่ม 1: แผนกผลิต
Production Shift Report Bot
  • ✅ สร้าง Claude Project อัปโหลด SOP + มาตรฐานการผลิต
  • ✅ สร้าง Dify Chatbot: ตอบปัญหา Machine troubleshoot
  • ✅ สร้าง n8n Workflow: สรุปรายงาน Shift ส่ง LINE Manager 17:00
🎯 CPF Shift Assistant — ตอบปัญหาสายการผลิตและสรุปรายงานอัตโนมัติ
🔬
กลุ่ม 2: แผนก QC
QC Alert + Root Cause Analyzer
  • ✅ Claude Project: มาตรฐาน QC ทุกผลิตภัณฑ์
  • ✅ n8n: Sheets → Claude วิเคราะห์ → LINE แจ้งด่วน
  • ✅ Claude: วิเคราะห์ root cause เมื่อพบ Lot ไม่ผ่าน
🎯 CPF QC Agent — ตรวจสอบอัตโนมัติและแจ้งเตือน 17:00 ทุกวัน
👥
กลุ่ม 3: แผนก HR
HR Policy Chatbot
  • ✅ Dify: Knowledge Base = HR Policy ทั้งหมด
  • ✅ ตั้งค่า System Prompt 'น้องซีพี'
  • ✅ Publish เป็น Web App + QR Code ติดบอร์ดประกาศ
🎯 น้องซีพี — ตอบคำถาม HR 24/7 สำหรับพนักงานโรงงาน
📦
กลุ่ม 4: Procurement
Stock Alert + PO Draft
  • ✅ n8n: ตรวจสต็อก Sheets ทุก 6 ชั่วโมง → แจ้ง LINE
  • ✅ Claude: ร่าง Purchase Order เมื่อสต็อกต่ำ
  • ✅ Dify: ตอบคำถาม Supplier และสินค้า
🎯 CPF Stock Agent — แจ้งเตือนและร่าง PO อัตโนมัติ
🛒
กลุ่ม 5: Sales/CS
Customer Complaint Handler
  • ✅ Make/n8n: รับ Email → Claude วิเคราะห์ → บันทึก Sheets
  • ✅ Dify: Product Catalog Chatbot สำหรับทีมขาย
  • ✅ Claude: ร่าง Email ตอบ Complaint อัตโนมัติ
🎯 CPF CS Agent — จัดการ Complaint อัตโนมัติและ Chatbot ผลิตภัณฑ์
