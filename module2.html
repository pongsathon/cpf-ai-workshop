สมัครและเข้าใช้ Claude.ai

1
สมัครบัญชี Claude.ai
ฟรี · ไม่ต้องใส่บัตรเครดิต
  • 1เปิด claude.ai ในเบราว์เซอร์
  • 2กด Sign up → Continue with Google
  • 3เลือก Google account → Allow
  • 4✅ เข้าสู่ Claude Dashboard
ใช้ Google Account ของบริษัท (@cpf.co.th) จะดีกว่า ข้อมูลแยกจากบัญชีส่วนตัว
2
สร้าง Project สำหรับ CPF
จำเอกสารได้ตลอด — ไม่ต้องอัปโหลดซ้ำ
  • 1กด Projects ในเมนูซ้าย
  • 2กด + New Project
  • 3ตั้งชื่อ: CPF QC Assistant
  • 4กด Create
claude.ai > Projects
📁 Projects ┌────────────────────────────────┐ │ 📂 CPF QC Assistant ← ใหม่ │ 📂 CPF HR Assistant │ 📂 CPF Feed Products └────────────────────────────────┘ [+ New Project]
3
อัปโหลดเอกสารเข้า Project
Claude จะ 'อ่านและจำ' เอกสารทั้งหมด
  • 1ใน Project กด Add content
  • 2เลือก Upload files
  • 3เลือกไฟล์ที่ต้องการ (PDF, Excel, Word, Markdown)
  • 4รอ Claude ประมวลผล (1-2 นาที)
  • 5✅ เห็นชื่อไฟล์ในรายการ = พร้อมใช้งาน
อัปโหลดได้สูงสุด 32 MB ต่อไฟล์ รองรับ PDF, Excel, Word, CSV, TXT, Markdown

เทคนิคการเขียน Prompt

4
โครงสร้าง Prompt ที่ดี
5 ส่วนที่ทำให้ได้ผลลัพธ์ที่ต้องการ
📋 โครงสร้าง Prompt สำหรับงาน CPF 
[บทบาท] คุณเป็น QC Expert ของ CPF

[ข้อมูล/เอกสาร] จากรายงาน QC ที่อัปโหลดไว้

[คำสั่ง] วิเคราะห์และสรุปผล

[รูปแบบ] แสดงเป็น bullet points แยก:
  🔴 ไม่ผ่าน | 🟡 ต้องระวัง | 🟢 ผ่าน

[ข้อจำกัด] ใช้ภาษาไทย ไม่เกิน 10 บรรทัด
5
เปรียบเทียบ Prompt ดี vs ไม่ดี
ความแตกต่างเล็กน้อยให้ผลต่างกันมาก
❌ Prompt ไม่ดี ✅ Prompt ที่ดีกว่า
สรุปให้หน่อย สรุปเป็น 3 bullet points เน้นปัญหา QC
วิเคราะห์ข้อมูล วิเคราะห์และระบุ LOT ที่ไม่ผ่าน พร้อม action
แปลเป็นไทย แปลเป็นไทย รักษาคำศัพท์เทคนิคและตัวเลข
เขียนอีเมล เขียนอีเมลทางการ ขอโทษ แจ้งวิธีแก้ ใต้ 100 คำ

Prompts สำเร็จรูปสำหรับ CPF

📋 Prompt: สรุปเอกสารยาว 
อ่านเอกสารที่แนบและสรุปดังนี้:

1. ประเด็นหลัก 3 ข้อ
2. ตัวเลขสำคัญ (ถ้ามี)
3. สิ่งที่ต้องดำเนินการ
4. คำถามที่ยังไม่ชัดเจน

ตอบเป็นภาษาไทย กระชับ ไม่เกิน 15 บรรทัด
📋 Prompt: ร่างอีเมลมืออาชีพ

รายละเอียด:
- ผู้รับ: [ตำแหน่ง]
- เรื่อง: [ระบุ]
- ข้อมูลสำคัญ: [ระบุ]
- Tone: [ทางการ/กึ่งทางการ]

ความยาวไม่เกิน 150 คำ ลงนาม [ชื่อ] ฝ่าย [แผนก]
