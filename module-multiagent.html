A1
Agent 1 — Data Collector (HTTP Request → Claude)
รับข้อมูลจาก Trigger แล้วให้ Claude สรุปข้อมูล LOT อย่างเป็นระบบ
  • 1กด + เพิ่ม Node → ค้นหา HTTP Request
  • 2Method: POST | URL: https://api.anthropic.com/v1/messages
  • 3Headers: x-api-key = Anthropic API Key, anthropic-version = 2023-06-01, content-type = application/json
  • 4Body: JSON → วาง Body ด้านล่าง (ใช้ n8n expression ดึงค่าจาก Trigger)

P1
Parse Output Agent 1 (Code Node)
แปลง string จาก Claude เป็น JSON object พร้อมส่งต่อ Agent 2
  • 1เพิ่ม Node: Code | Mode: Run Once for Each Item
  • 2วาง code ด้านล่าง
⚙️ Code Node — Parse Claude Output
const raw = $input.item.json.content[0].text
  .replace(/^```json\s*/i,'').replace(/```\s*$/,'').trim();
const agent1 = JSON.parse(raw);
// ส่งต่อทั้ง agent1 output + ข้อมูล original
return {
  json: {
    agent1_output: agent1,

P2
Parse Output Agent 2 + รวมข้อมูลทั้งหมด
รวม agent1 + agent2 output เป็น object เดียว พร้อมส่ง Agent 3
⚙️ Code Node — Merge agent1 + agent2
const raw = $input.item.json.content[0].text
  .replace(/^```json\s*/i,'').replace(/```\s*$/,'').trim();
const agent2 = JSON.parse(raw);
return {
  json: {
    agent1_output: $input.item.json.agent1_output,  // จาก P1 (pass-through)
    agent2_output: agent2
  }
};
⚠️ HTTP Request node ส่งแค่ response จาก Claude กลับมา — ข้อมูลจาก node ก่อนหน้าหายไป ต้องใช้ Code Node รวม (merge) ด้วยตัวเอง

S3
Merge + Parse — รวม output จาก 1A และ 1B
รอให้ทั้งสอง branch เสร็จแล้วรวม → ส่งต่อ Agent 2
  • 1เพิ่ม Node: Merge
  • 2Mode: Wait for All Input Streams ← สำคัญมาก ต้องรอทั้งสอง branch
  • 3ลาก connection จาก Agent 1A และ Agent 1B มาที่ Merge Node
  • 4หลัง Merge เพิ่ม Code Node เพื่อ parse และรวมข้อมูล
⚙️ Code Node — Parse + Merge 1A & 1B
// Merge node ส่ง items มาเป็น array [item1A, item1B]
const items = $input.all();

function parseAgent(item) {
  const raw = item.json?.content?.[0]?.text ?? '';
  try {
    return JSON.parse(raw.replace(/^```json\s*/i,'').replace(/```\s*$/,'').trim());
  } catch { return { raw }; }
}

const agent1a = parseAgent(items[0]);  // QC Checker
const agent1b = parseAgent(items[1]);  // Policy Checker

return {
  json: {
    qc_result: agent1a,
    policy_result: agent1b
  }
};

⚠️ ข้อควรระวัง

❌ ที่มักทำผิด
  • ไม่ parse JSON ก่อนส่งต่อ Agent ถัดไป
  • ลืม merge ข้อมูล Agent ก่อนหน้า (ข้อมูลหาย)
  • Merge Node ใน Parallel ลืมเปิด "Wait for all streams"
  • ใส่ข้อมูล context มากเกินไปใน prompt → token เกิน
✅ Best Practice
  • Test ทีละ Node ก่อน connect
  • ให้แต่ละ Agent ตอบ JSON เสมอ — parse ง่ายกว่า
  • ตั้งชื่อ Node ให้ชัด: "HTTP Agent 1A QC" ไม่ใช่ "HTTP Request"
  • ใส่ error handling (Code Node try-catch) ทุก parse step
💡 เริ่มจาก Workflow 1 ก่อน — Sequential ง่ายกว่า เข้าใจ flow ก่อน แล้วค่อยต่อ Workflow 2 ที่มี Parallel
